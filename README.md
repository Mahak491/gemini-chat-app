# Gemini Chat App

A full-stack chat application powered by **Google's Gemini AI API**. This app demonstrates how to integrate cutting-edge AI capabilities into a web application with a modern, responsive user interface.

## ✨ Features

- **Real-time AI Chat** - Get instant responses from Google's Gemini AI model
- **Modern UI/UX** - Clean, gradient-based design with smooth animations
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Loading States** - Visual feedback while waiting for AI responses
- **Error Handling** - Graceful error handling with user-friendly messages
- **Dark Mode Support** - Automatically adapts to system preferences
- **Keyboard Support** - Press Enter to send messages quickly

## 🛠️ Tech Stack

### Frontend
- **React 18** - UI library for building the chat interface
- **CSS3** - Custom styling with animations and gradients
- **Fetch API** - For making HTTP requests to the backend

### Backend
- **Node.js** - JavaScript runtime environment
- **Express.js** - Web framework for building REST APIs
- **Google GenAI SDK** - Official Google SDK for Gemini API integration
- **CORS** - Cross-Origin Resource Sharing enabled for security

### AI Model
- **Gemini 2.5 Flash** - Google's fast and efficient language model

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js** (v14 or higher) - [Download](https://nodejs.org/)

- **Git** (optional, for cloning)

You'll also need:
